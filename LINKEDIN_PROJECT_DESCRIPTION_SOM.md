# HillNest Project Description

Built HillNest, a full-stack mountain homestay booking platform, with a strong focus on secure product engineering and production readiness. As a full-stack collaborator and security engineer, I worked across the customer app, backend services, and admin platform to turn the system from a feature build into a safer, more reliable application.

My contributions included migrating customer authentication to `httpOnly` cookie sessions, adding refresh-token handling, strengthening backend protections with `Helmet`, `trust proxy`, rate limiting, and request limits, and tightening API response shaping so sensitive user data stays hidden. I also added validation and input safety for room filters, bookings, and payment verification, built security regression tests with saved logs, and helped stabilize the Docker-based local environment and admin service boot flow.

On the frontend, I cleaned up admin login, dashboard, monitoring, and shared table code, and added dev-time room seeding so the Explore Rooms flow works with real data locally. I also supported safer auth boundaries between customer and admin flows, plus better session handling across the app.

Tech Stack: Next.js 15, TypeScript, React, Node.js, Express, MongoDB, Mongoose, PostgreSQL, Prisma, Redis, Docker, JWT, RBAC, `httpOnly` cookies, Helmet, and Node test runner.

This project let me blend full-stack delivery with practical security engineering: safer auth flows, stronger API defense, cleaner operations, and a more contributor-friendly codebase.
